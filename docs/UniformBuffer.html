Uniform Buffer

class UniformBuffer

help manage the uniform buffer creation and updating their GPU memory. This class is mainly a dictionary of variable of different type that can be send to the the GPU.

Public Functions

template<typename T>
inline void addVariable(const std::string &name, const T &value)

Add a variable into the dictionary.

Parameters

  • name – the name of constant.

  • value – the variable, either a simple data type of a contiguous range of simple data type.
