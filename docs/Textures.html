Image LavaCake::Framework::createTextureBuffer(const Queue &queue, CommandBuffer &cmdBuff, const std::string &filename, int nbChannel, VkFormat f = VK_FORMAT_R8G8B8A8_UNORM, VkPipelineStageFlagBits stageFlagBit = VK_PIPELINE_STAGE_FRAGMENT_SHADER_BIT)

create an image specialized to be a texture buffer and initialize it with the data contained in a file.

Parameters

  • queue – a pointer to the queue that will be used to copy data to the Buffer

  • cmdBuff – the command buffer used for this operation, must not be in a recording state

  • filename – the path to the file containing the data

  • nbChannel – the number of channel to use from the image

  • format – the format of the image

  • stageFlagBit – the stage in which the shader will be used
