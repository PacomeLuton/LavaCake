create an image specialized to be a cubemap and initialize it with a set of texture.

Parameters

  • queue – a pointer to the queue that will be used to copy data to the Buffer

  • cmdBuff – the command buffer used for this operation, must not be in a recording state

  • filename – the path to the folder containing the textures

  • images – the names of the textures

  • nbChannel – the number of channel to use from the textures

  • format – the format of the image

  • stageFlagBit – the stage in which the shader will be used
